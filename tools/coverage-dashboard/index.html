Coverage Breakdown

A read-only window onto the project's coverage gate — the same numbers and thresholds CI enforces. This page shows the data; it never changes it.

🔭 Scope: leaderboard/shared/ only. This gate measures the deterministic, framework-free leaderboard logic — the one place in this repo where a coverage percentage is meaningful. It is not a whole-repository coverage gate: games, the marketing site, DB adapters, dev servers and tooling are deliberately out of scope. Thresholds live in leaderboard/.c8rc.json and change only through a code-reviewed config edit — never from this page.
Optional: inspect a local coverage-summary.json from npm run coverage. Viewing only — nothing is saved or uploaded.
